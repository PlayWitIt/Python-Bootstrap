PyBootstrap

Quick setup for Python projects. One command, then start coding.

Install

pip install PythonBootstraper

Quick Start

# Create project with both scripts
pybs init

# Or in a new folder
pybs init --dir myproject

# Run scripts directly
pybs exec myscript.py

Commands

  • pybs init - Creates py_bootstrap.sh + Run.sh
  • pybs bootstrap - Creates only py_bootstrap.sh
  • pybs run - Creates only Run.sh
  • pybs exec script.py - Run a script directly

Options

  • --dir name - Create in a new directory
  • --force - Overwrite existing files
  • --no-venv - Use system Python (with exec)

How it works

  1. Run pybs init to create the scripts
  2. Run ./py_bootstrap.sh once to set up your environment
  3. Run ./Run.sh (or pybs exec) to run your code

What py_bootstrap.sh does

  • Detects Python (pyenv, conda, or system)
  • Creates virtual environment
  • Installs dependencies from requirements.txt
  • Offers to set up Git

What Run.sh does

  • Finds your virtual environment
  • Activates it
  • Runs your Python script (auto-detects main.py, app.py, etc)

Example

mkdir myproject
cd myproject
pip install PythonBootstraper
pybs init
echo 'print("Hello!")' > main.py
./py_bootstrap.sh
./Run.sh
